### API Endpoints: 
- Adding a restaurant and getting all restaurants: 
GET, POST: <host url>/v1/restaurants/

- Relating to individual restaurants specified by primary key: 
GET, PUT, DELETE: <host url>/v1/restaurants/<primary key>/
